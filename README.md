# ScreenScraps

> A beautiful MacOS Utility to clean up your screenshots!

Built by Hallie

Simple MacOS menu bar app which allows you to clean up your desktop by hiding screenshots in the menu bar! 

### Features

* Drag and drop!
* Auto-copy to clipboard
* Rename screenshots
